这是一个在线考试管理系统，基于 Java EE（Struts2 + Spring + MyBatis）架构。主要功能：                                                                                                
1. 题库管理 — 支持单选、多选、填空等题型，按知识点（KeyPoint）分类，可设置难度等级                                                                      
2. 考试策略配置 — 定义出卷规则（题型配比、难度分布、题目来源范围），支持随机组卷和指定组卷两种模式                                                      
3. 试卷管理 — 按策略自动生成试卷，支持手动调整，定时自动生成                                                                                            
4. 在线考试 — 考生登录后根据分配的考试范围参加考试，实时记录答题和监控状态                                                                              
5. 阅卷与评分 — 主观题人工阅卷，客观题自动判分，支持成绩查询和统计分析                                                                                  
6. 权限管理 — 基于 RBAC 的用户/角色/权限体系，集成 CAS 单点登录
